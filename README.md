# Daughter of Zion Catering

A modern, responsive catering website built for **Daughter of Zion**, a Liberian catering business specializing in authentic homemade cuisine for weddings, corporate events, private parties, and special occasions.

The website focuses on showcasing Liberian culture through elegant design, rich photography, and a seamless customer experience for browsing the menu and requesting catering services.

---

## Project Goals

- Showcase authentic Liberian cuisine
- Promote catering services
- Display menu items and featured dishes
- Allow customers to submit booking requests
- Build trust through testimonials and gallery
- Provide a modern, mobile-friendly experience

# Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/daughter-of-zion-catering.git
```

Navigate into the project

```bash
cd daughter-of-zion-catering
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

# Design Principles

The design is inspired by:

- Authentic Liberian culture
- Warm hospitality
- Premium catering services
- Elegant typography
- Earth-tone color palette

- Rich food photography

The goal is to create a luxury yet welcoming digital experience that reflects the warmth of Liberian cuisine.
